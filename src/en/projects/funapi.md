A minimal, async-first Ruby web framework inspired by FastAPI. Built on top of Falcon and dry-schema, FunAPI provides a simple, performant way to build web APIs in Ruby with a focus on developer experience. Async-first with Ruby's Async library, simple validation with dry-schema, minimal magic, and automatic OpenAPI/Swagger documentation generation.
